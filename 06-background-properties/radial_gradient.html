Radial Gradient

Radial gradients are a type of gradient that transition smoothly between two or more colors in a circular or elliptical pattern. They are often used to create visually appealing backgrounds, buttons, and other elements in web design and graphic design. The center of the gradient can be specified, allowing for different focal points, and the shape can be adjusted to create circular or elliptical gradients. Radial gradients can also include multiple color stops, enabling designers to create complex color transitions. In CSS, radial gradients can be applied using the `radial-gradient()` function, which takes color values and optional shape and position parameters to define the gradient's appearance.

Example of a radial gradient in CSS:

background: radial-gradient(circle, #ff7e5f, #feb47b);
