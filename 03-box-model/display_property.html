Display Properties

Defination: Display properties are used to control how an element is displayed on the web page. The most common display properties include:

  • block: Displays the element as a block-level element.
  • inline: Displays the element as an inline-level element.
  • inline-block: Displays the element as an inline-level block container.
  • none: Hides the element from the page.
  • visibility hidden: Hides the element with element space from the page.
