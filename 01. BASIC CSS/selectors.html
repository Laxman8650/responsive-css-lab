Selectors in CSS

CSS selectors are used to select the HTML elements that you want to style. There are different types of selectors in CSS, including:

  • Element Selector - Direct selection of HTML elements like h1, h2, p, div
  • Class Selector - .class-name.(.symbol in CSS to select an element with that class),(Reusable)
  • ID Selector - #id-name (unique identifier),(Use of # symbol in CSS to select an id)
  • Attribute Selector - [attribute] like [href], [title]
  • Pseudo-class Selector - :hover, :focus, etc.
  • Pseudo-element Selector - ::first-line, ::first-letter, etc.
  • Universal Selector - * (selects all elements)
