Units of CSS

In CSS, there are various units of measurement that can be used to specify the size, length, and other properties of elements. These units can be categorized into two main types: Absolute units and Relative units.

Absolute Units

  • px (pixels): A pixel is a unit of measurement that represents a single dot on the screen. It is commonly used for defining fixed sizes.
  • cm (centimeters): A centimeter is a unit of measurement that represents 1/100th of a meter. It is used for print media and physical measurements.
  • mm (millimeters): A millimeter is a unit of measurement that represents 1/1000th of a meter. It is also used for print media and physical measurements.
  • in (inches): An inch is a unit of measurement that represents 2.54 centimeters. It is commonly used in the United States for print media and physical measurements.

Relative Units

  • em: The em unit is relative to the font size of the parent element. For example, if the font size of the parent element is 16px, then 1em would be equal to 16px.
  • rem: The rem unit is relative to the font size of the root element (usually the <html> element). For example, if the font size of the root element is 16px, then 1rem would be equal to 16px.
  • % (percentage): The percentage unit is relative to the size of the parent element. For example, if a width is set to 50% and the parent element has a width of 200px, then the width of the child element would be 100px.
  • vw (viewport width): The vw unit is relative to the width of the viewport. For example, if an element has a width of 50vw and the viewport is 1000px wide, then the width of the element would be 500px.
  • vh (viewport height): The vh unit is relative to the height of the viewport. For example, if an element has a height of 50vh and the viewport is 800px tall, then the height of the element would be 400px.
