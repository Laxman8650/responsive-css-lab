Position Properties

It determines how an element is placed on a web page and how it interacts with the standard document flow.

Types of Positioning

  • Static Positioning: This is the default positioning for all elements. Elements are positioned according to the normal flow of the document.
  • Relative Positioning: The element is positioned relative to its normal position. You can use the top, right, bottom, and left properties to adjust its position.
  • Absolute Positioning: The element is positioned relative to its nearest positioned ancestor (not static). If there is no such ancestor, it is positioned relative to the initial containing block (usually the viewport).
  • Fixed Positioning: The element is positioned relative to the viewport, which means it stays in the same place even when the page is scrolled. Used for elements like navigation bars or "back to top" buttons.
  • Sticky Positioning: The element toggles between relative and fixed positioning based on the user's scroll position. It behaves like relative until a certain scroll point is reached, then it becomes fixed.
